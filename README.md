# SuperMart-Grocery-Sales-Analysis-Dashboard
Data analysis project using Excel to track sales, profit, and trends with an interactive dashboard and business insights.


### 🛒 SuperMart Grocery Sales Analysis Dashboard (Excel Project)

### 📌 Project Overview

This project analyzes SuperMart grocery sales data to uncover key business insights across categories, regions, and time. The objective is to transform raw data into actionable insights using Excel and support better decision-making.

### 🎯 Problem Statement

Retail businesses often struggle to identify high-performing categories, profitable regions, and seasonal trends. This project aims to provide a clear and interactive dashboard to track performance and improve business strategy.

### 📂 Dataset Description

The dataset contains transactional sales data, including category, sub-category, region, sales, profit, and order details.
Data was cleaned and structured before analysis to ensure accuracy.

### 🧹 Data Cleaning Steps
* Removed duplicate records
* Handled missing values
* Standardized date formats
* Structured columns for analysis

### 📊 Key Metrics
* Total Sales: ₹1,49,56,982
* Total Profit: ₹37,47,121
* Average Profit Margin: 25.02%
* Average Order Value: ₹1,497

### 📈 Analysis Performed
* Category-wise sales and profit analysis
* Region-wise performance comparison
* Monthly trend analysis to identify seasonality
* Sub-category level performance insights

### 📊 Dashboard Features
* KPI Cards (Sales, Profit, Margin, AOV)
* Sales by Category and Sub-Category
* Region-wise Performance
* Monthly Sales Trend
* Interactive Slicers for filtering

## 💡 Key Insights

### 🏆 Category Performance
* Egg, Meat & Fish is the top-performing category by sales
* Snacks generate the highest profit
* Oil & Masala is the lowest-performing category

### 🌍 Regional Performance

* West region leads in both sales (₹47,98,743) and profit (₹11,92,005)
* The North region shows negligible sales and profit
* Other regions show steady performance

### 📅 Sales Trends

* Sales peak in September, November, and December
* February records the lowest sales

### 🧃 Sub-Category Insights

* Health Drinks and Soft Drinks generate the highest sales

### 🧠 Advanced Insights

* All categories generate similar sales (₹20–22 lakh) and profit (₹4.9–5.9 lakh), indicating a balanced product portfolio
* Despite the highest sales, the West region has relatively weaker profit margins
* High-volume sub-categories do not always convert into proportional profit
* The North region shows zero/very low sales, indicating a major business gap

### 🚀 Business Recommendations

* Improve pricing strategy in the West and East regions
* Investigate issues in the North region
* Focus on the Snacks category for profit growth
* Leverage seasonal trends (Sep–Dec)
* Optimize high-volume sub-categories for better profit
